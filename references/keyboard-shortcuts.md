## Tips

### Quick Toggle
`Cmd + E` is your most used shortcut. Use it to:
- Check agent progress (Manager view)
- Write code (Editor view)
- Go back and forth as needed

### Inline Completions
`Cmd + I` opens inline prompt anywhere in code:
- Select code → `Cmd + I` → "Refactor this"
- Empty line → `Cmd + I` → "Write a function that..."

### Terminal Integration
In terminal, select output then:
- `Cmd + L` sends it to agent for analysis
